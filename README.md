# Task_2_Adam-sherif
Overview

This project was completed as part of the Data Analytics Industrial Training Program at DecodeLabs. The objective of the project is to perform Exploratory Data Analysis (EDA) on an e-commerce dataset to uncover patterns, trends, and valuable business insights.

Objectives
Clean and prepare the dataset for analysis.
Explore sales and order patterns.
Identify trends and customer behavior.
Detect potential outliers.
Create visualizations to support findings.
Generate business insights from the data.
Tools Used
Microsoft Excel
Power Query
Pivot Tables
Pivot Charts
Conditional Formatting
Data Cleaning Process

The following steps were performed:

Checked for missing values.
Reviewed duplicate records.
Standardized data types.
Extracted Year and Month from order dates using Power Query.
Validated data consistency across columns.
Analysis Performed
Sales Analysis
Total Sales by Product
Monthly Sales Trends
Yearly Sales Performance
Order Analysis
Distribution of Order Statuses
Delivered vs Returned vs Cancelled Orders
Customer & Marketing Analysis
Payment Method Performance
Customer Purchasing Patterns
Key Insights
Identified top-performing products based on sales.
Analyzed the most common order statuses
Discovered trends in sales performance over time.
Highlighted unusual transactions that may require further investigation.

Conclusion

This project demonstrates the use of Excel and Power Query for Exploratory Data Analysis (EDA). Through data cleaning, visualization, and business analysis, meaningful insights were extracted from the dataset to support data-driven decision-making.
